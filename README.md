# wetube_reload Router

<!-- global router -->

/ -> home
/login -> login
/join -> join
/search -> search

<!-- user router -->

/user/:id -> see user
/user/edit -> edit user
/user/delete -> delete user
/user/logout -> logout user

<!-- video router -->

/video/:id -> watch video
/video/:id/edit -> edit video
/video/:id/delete -> delete video
/video/upload -> upload video
